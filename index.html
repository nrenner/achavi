Usage
  • zoom in to your area of interest
  • specify time range of changes to load, check to include relations, press 'load'.
    Attic data goes back until September 2012.
  • For continuous monitoring bookmark the Permalink and simply press load each time,
    the last load date will be preset as start date.
  • A single changeset can be visualized by drag & drop of a changeset URL or using a bookmarklet, see Wiki page.
  • hover over feature to show popup, click on feature to keep popup open, click on map to close
  • zoom on changes with shift + drag zoom box

about
achavi - Augmented OSM Change Viewer  [attic]

Displays OpenStreetMap changes based on the Overpass API Augmented Delta (adiff) query and the Augmented Diff format.
Tested on Firefox and Chrome; older Internet Explorer and mobile not supported.

This is an early alpha version and many features are missing, like better relation support, changeset
information, textual list and/or summary of changes and performance optimization. Also expect to find bugs.
